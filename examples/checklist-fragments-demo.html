SmartArt Checklist

with Reveal.js Fragments

Progressive reveal and auto-check demonstration

Configuration Options

Available Options:

  • fragments: Enable/disable fragments (default: true)
  • fragmentAnimation: Animation style (default: "fade-up")
  • autoCheck: Auto-check on reveal (default: true)

Fragment Animations:

  • fade-up, fade-down, fade-left, fade-right
  • grow, shrink
  • fade-in-then-out, current-visible

Usage Example


<div data-smartart='{
    "layout": "checklist",
    "heading": "Key Learnings",
    "intro": "Key learnings from this section:",
    "fragmentAnimation": "fade-up",
    "items": [
        {
            "title": "First learning point"
        },
        {
            "title": "Second learning point"
        },
        {
            "title": "Third learning point",
            "checked": true
        }
    ]
}'></div>
