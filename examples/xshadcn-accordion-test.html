Accordion Props

Available Properties
  • type: "single" | "multiple"
    Controls whether one or multiple items can be open
  • defaultValue: string | string[]
    Initially expanded item(s)
  • items: Array<{value, title, content}>
    Array of accordion items to display
  • className: string
    Custom Tailwind CSS classes for styling
  • componentId: string
    Unique identifier for state management
