What is LLM Runner?

A reveal.js plugin that brings interactive coding and AI chat to your presentations.

  • Execute Python code live in slides
  • Chat with Claude and GPT models
  • Real-time streaming responses
  • Context persistence per slide
  • Keyboard shortcuts for speed

Execute Python code with real-time output

print("Hello from LLM Runner!")

print(f"Circle area (r={radius}): {area:.2f}")

Demo 2: Data Analysis

Generate and analyze data on the fly

import random
from collections import Counter

# Generate random dice rolls
rolls = [random.randint(1, 6) for _ in range(100)]

# Analyze frequency
counter = Counter(rolls)
print("Dice Roll Distribution (100 rolls):\n")

for number, count in sorted(counter.items()):
    bar = '█' * (count // 2)  # Visual bar chart
    print(f"{number}: {bar} ({count})")

Demo 7: Advanced Python

Complex calculations and algorithms

# Fibonacci sequence generator
def fibonacci(n):
    """Generate Fibonacci sequence"""
    fib = [0, 1]
    for _ in range(n - 2):
        fib.append(fib[-1] + fib[-2])
    return fib

# Generate and display
fib_sequence = fibonacci(15)
print(f"First 15 Fibonacci numbers:")
print(fib_sequence)

# Golden ratio approximation
ratio = fib_sequence[-1] / fib_sequence[-2]
print(f"\nGolden ratio approximation: {ratio:.6f}")

Quick Setup

  1. Install Python dependencies: 
    pip install -r plugin/llm-runner/requirements.txt
  2. Create .env file with API keys: 
    ANTHROPIC_API_KEY=your_key
OPENAI_API_KEY=your_key
  3. Start the server: 
    python plugin/llm-runner/server.py
  4. Add plugin to reveal.js: 
    plugins: [ RevealLLMRunner ]
