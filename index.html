Spotify Web API

A REST interface to search the Spotify catalog

Search & Lookup

http://ws.spotify.com/service/version/method{.format}?parameters

http://ws.spotify.com/search/1/album.json?q=foo
http://ws.spotify.com/search/1/artist.json?q=foo
http://ws.spotify.com/search/1/track.json?q=foo

http://ws.spotify.com/lookup/1/.json?uri={uri}
  • Supports JSON (and XML...)
  • Advanced search syntax
  • UTF-8 responses
  • Territory availability
  • Caching

Apps inside Spotify

Spotify Apps API

  • Chromium Embedded Framework
  • HTML 5 support: canvas, drag & drop, geolocation, WebGL, local storage, web workers, sockets
  • Play music, access and modify playlists, interact with user data
  • Beta version of async 1.0 Javascript API.
  • Beta version of Web Player SDK: (Code | Docs)
  • Spotify Apps Tutorial: an app that shows you how to make an app. Branch 1.0 uses the asyncronous API.
